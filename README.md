# Lappy_Piano
Web app that allows you to play piano using your laptop keyboard. Try it out!! https://tsarim.github.io/Lappy_Piano/
<img src="finalApp.png" alt="">
<br>
<h3>How to use?</h3>
<h4>The following mapping is used:</h4>
<p>key A -> note C3</p>
<p>key S -> note D3</p>
<p>key D -> note E3</p>
<p>key F -> note F3</p>
<p>key G -> note G3</p>
<p>key H -> note A3</p>
<p>key J -> note B3</p>
<p>key K -> note C4</p>
<p>key L -> note D4</p>
<p>key ; -> note E4</p>
<p>key N -> note F4</p>
<p>key M -> note G4</p>
<p>key , -> note A4</p>
<p>key . -> note B4</p>
